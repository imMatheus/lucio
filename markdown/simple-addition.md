# Simple addition

### Problem description

Given two integers, **a** and **b**, return the sum of **a** + **b**

### Input format

**a** a integer, **b** a integer

### Output

int: the sum of **a** + **b**

### Constrains

-   -1000 < **a** < 1000
-   -1000 < **b** < 1000

### Sample inputs

```bash
9
6
```

### Sample output

```bash
15
```

### Explanation

9 + 6 = 15

### Sample inputs

```bash
108
12
```

### Sample output

```bash
120
```
