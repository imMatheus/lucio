# Multiplication

### Problem description

Given two numbers, **x**, **y**, return the product of **x** \* **y**

### Input format

**x** a integer, **y** a integer

### Output

int: the product of **x** \* **y**

### Constrains

-   -1000 < **x** < 1000
-   -1000 < **y** < 1000

### Sample inputs

```bash
30
3
```

### Sample output

```bash
90
```

### Explanation

30 \* 3 = 90
